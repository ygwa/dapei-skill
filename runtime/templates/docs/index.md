## Getting Started

To explore the durable knowledge and cognitive assets of your workspace:

1. Use `@dapei` in your AI chat to initialize, add repositories, or analyze behaviors.
2. Run VitePress locally to view live updates:
   ```bash
   pnpm run docs:dev
   ```
3. When requirements are implemented, closing the feature via `@dapei close feature` will automatically backfill verified design decisions here.
